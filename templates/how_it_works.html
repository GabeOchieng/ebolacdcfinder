How it works

  1. The Ebola CDC finder is a tool that helps people to register or find Ebola centers for disease control and prevention in their neighborhood.


  2. A person that needs an Ebola CDC specifies the distance within which S/he expects the Ebola CDC and then clicks search Ebola CDC.


  3. The tool then returns the results basing on the distance selected by the user and the current position of the user. The user can select any item of the returned results to view more details about any Ebola CDC result returned.


  4. Each result contains the information like the location of the Ebola CDC, the Ebola CDC name,contact which can be used by the anyone to get help or make an inquiry.

  5. The tool also enables registration of an Ebola CDC in their area by clicking on the map where the Ebola CDC is located (A blue pin will apear on the location clicked by the user) and also specifying other related information to the location of the Ebola CDC.

Note:

This tool uses geolocation and therefore it should be enabled in your web browser.
